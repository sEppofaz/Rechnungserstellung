Wie funktioniert die App?

Handgeschriebenen Rechnungszettel fotografieren → Claude liest alle Felder aus → du prüfst und korrigierst → Rechnung wird als ODT-Datei in Dropbox (Rechnungen_Entwurf/) abgelegt.

Zugangscode

Einmalig beim ersten Start eingeben. Der Code wird gespeichert – danach kein erneutes Einloggen nötig.

Ausgabe

Fertige Rechnung liegt in Dropbox unter Rechnungen_Entwurf/ als .odt-Datei.
